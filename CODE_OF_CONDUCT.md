# Code of Conduct

## Our Standard

Be respectful, practical, and specific. Disagreement is fine; harassment, personal attacks, and bad-faith behavior are not.

## Unacceptable Behavior

- harassment or discriminatory language
- publishing private information
- threats or intimidation
- intentionally derailing technical discussion
- encouraging abuse of sites, services, or access controls

## Enforcement

Maintainers may edit, hide, or remove comments, issues, pull requests, or access for behavior that violates this code of conduct.
