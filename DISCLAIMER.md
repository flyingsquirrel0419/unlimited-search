# Disclaimer

`unlimited-search` is a tool for reading public web content.

It does not provide permission to access content. Users are responsible for complying with applicable laws, site terms, robots policies, rate limits, and contractual obligations.

Fallback results may come from public APIs, RSS/Atom feeds, Jina Reader, metadata embedded in HTML, or media metadata providers. Those results can be incomplete, stale, transformed, or different from a fully rendered browser page.

The project is not intended to:

- bypass logins
- bypass paywalls
- defeat CAPTCHA or human-verification systems
- access private networks
- collect credentials
- impersonate a user account
- bypass IP bans or hard anti-abuse systems

If a site requires authentication or payment, `unlimited-search` should stop and report that access is required.
